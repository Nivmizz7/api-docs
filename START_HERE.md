# START HERE - Tarkov API Documentation

Complete documentation for the Tarkov API is now ready for use.

## Quick Start (30 seconds)

```bash
# Install dependencies
pip install mkdocs mkdocs-material

# Start documentation server
mkdocs serve

# Open in browser
http://localhost:8000
```

## Deployment Options

### Option 1: Local Viewing
```bash
mkdocs serve
```

### Option 2: Static Site
```bash
mkdocs build
# Deploy 'site/' folder to any web server
```

### Option 3: GitHub Pages
```bash
pip install ghp-import
mkdocs build
ghp-import -p site
```
